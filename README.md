# Trader Behavior vs. Market Sentiment Analysis

## Project Overview
This project analyzes how Bitcoin market sentiment (Fear vs. Greed) impacts trader behavior and profitability on Hyperliquid. The goal is to identify patterns that can inform risk management and trading strategies.

## Data Sources
1. [cite_start]**Bitcoin Market Sentiment:** Daily Fear & Greed index values[cite: 13].
2. [cite_start]**Historical Trader Data:** Transaction-level data including price, size, and PnL[cite: 16].
